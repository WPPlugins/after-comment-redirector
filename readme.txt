Redirect to custom page after commenting for all or new commentators.

== Description ==

This plugin can redirect a commentator to a specific URL. This URL can be set on the settings page. You have two different options: You can either redirect after every comment or just the ones of commentators who have no approved comment. This is recognized by their email address.

The function of this plugin can be used as a marketing tool. You can create a thank you page, but you can also advertise for example your newsletter or your social media pages.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/after-comment-redirector' directory or install it trough the WordPress plugin screen directly.
2. Activate the plugin.
3. Configure the URL on the Settings->'After Comment Redirector' page.
4. Save your settings.
